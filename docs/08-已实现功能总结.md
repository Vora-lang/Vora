## 语言特性总览

### 数据类型

| 类型 | Vora 字面量 | 内部表示 | 示例 |
|------|------------|----------|------|
| Null | `null` | `std::nullptr_t` | `let x = null` |
| Int | 整数 | `int64_t` | `42`, `0xFF`, `0o77`, `0b101` |
| Float | 浮点数 | `double` | `3.14`, `1e10` |
| Boolean | `TRUE` / `FALSE` | `bool` | `let flag = TRUE` |
| String | 双引号字符串 | `std::string` | `"hello"` |
| Array | `[...]` | `shared_ptr<Array>` | `[1, 2, 3]` |
| Dict | `{key: val, ...}` | `shared_ptr<Dict>` | `{name: "Vora", year: 2026}` |
| Function | `func` 声明 | `shared_ptr<VoraFunction>` | `func f() {}` |
| Object | `Obj` 声明 | `shared_ptr<ObjectInstance>` | `Obj Foo() {}` |

### 变量系统

```vora
let x = 10              // 声明并初始化
x = 20                  // 重新赋值
```

- 使用 `let` 关键字声明
- 支持块级作用域
- 变量沿作用域链向上查找

### 运算符

| 分类 | 运算符 | 说明 |
|------|--------|------|
| 算术 | `+`, `-`, `*`, `/`, `%` | 加、减、乘、除、取模 |
| 幂运算 | `**` | 右结合 |
| 比较 | `<`, `<=`, `>`, `>=` | 关系比较 |
| 相等 | `==`, `!=` | 相等/不等 |
| 逻辑 | `&&`, `||`, `!` | 与、或、非 |
| 自增/自减 | `++`, `--` | 前缀/后缀均支持 |
| 复合赋值 | `+=`, `-=`, `*=`, `/=`, `%=` | 加法/减法/乘法/除法/取模赋值 |
| 三元 | `? :` | 条件三元运算符，短路求值，右结合 |
| 赋值 | `=` | 变量赋值和属性赋值 |

### 字符串插值

```vora
let name = "World"
print("Hello ${name}!")          // 输出: Hello World!

Obj Person(n) { this.name = n }
let p = Person("Alice")
print("Hi ${p.name}")            // 输出: Hi Alice
```

- 语法：`${variable}` 或 `${obj.property}`
- 支持多级属性访问
- 自动类型转换（数字、布尔、null → 字符串）
- 变量未找到时保留原始 `${...}` 文本

### 字符串拼接与索引

```vora
let s = "Hello" + " " + "World"  // 字符串拼接
print(s[0])                       // 索引访问 → "H"
print(s[3])                       // → "l"
```

- `+` 运算符支持 `string + string` 拼接
- `[]` 索引访问单个字符（返回字符串）
- 字符串可用单引号或双引号

### 类型标注

```vora
let x:int = 42                   // 标注期望 int 类型
let y:float = 3.14               // 标注期望 float 类型
let z = int(input("> "))         // 类型转换 + 输入
```

- `:int` / `:float` 标注（当前仅作文档用途，不强制校验）
- 配合 `int()` / `float()` 进行显式类型转换

### 控制流

#### if / else

```vora
if (x > 0) {
    print("positive")
} else if (x < 0) {
    print("negative")
} else {
    print("zero")
}
```

#### 三元运算符 `? :`

```vora
let result = x > 0 ? "positive" : "negative"

// 右结合：a ? b : c ? d : e  →  a ? b : (c ? d : e)
let grade = score > 90 ? "A" : score > 80 ? "B" : "C"

// 短路求值：只求值被选中的分支
let val = condition ? expensiveOp1() : expensiveOp2()
```

#### while 循环

```vora
let i = 0
while (i < 10) {
    print(i)
    i += 1                       // 复合赋值
}
```

- 支持 `break` / `continue`

#### for-in 循环

```vora
// 数组
for item in [1, 2, 3, 4, 5] {
    print(item)
}

// 字符串
for ch in "Vora" {
    print(ch)
}

// range
for i in range(5) {
    print(i)
}
```

- 支持数组、字符串、`range()` 结果、字典、对象（遍历自有属性键）
- 每次迭代创建新作用域
- 支持 `break` / `continue`

### 函数

```vora
func add(a, b) {
    return a + b
}

let result = add(1, 2)    // result = 3
```

- 一等公民（可作为值传递）
- 闭包支持（捕获定义时的环境）
- 参数个数校验
- 通过 `return` 返回值（无 return 则返回 null）

#### 默认参数

```vora
func greet(name, greeting = "Hello") {
    return greeting + ", " + name
}
greet("World")             // → "Hello, World"
greet("Vora", "Hi")        // → "Hi, Vora"

// 默认值可引用前置参数
func addOne(a, b = a + 1) {
    return b
}
addOne(5)                  // → 6
addOne(5, 10)              // → 10

// 全默认参数
func allDefaults(a = 1, b = 2, c = 3) { return a + b + c }
allDefaults()              // → 6
allDefaults(10)            // → 15
```

- 默认参数在调用时可选，未传递时使用默认表达式求值
- 默认表达式可引用该参数之前的任何形参
- 编译为 `OP_DEFAULT_PARAM` 操作码，运行时高效 skips

### 数组

```vora
let arr = [1, "hello", TRUE, null]
print(arr[0])              // 输出: 1
print(arr[1])              // 输出: hello
arr += [3, 4]              // 数组拼接

for item in arr {          // 遍历
    print(item)
}
```

- 动态类型元素
- 整数索引（从 0 开始）
- 越界检查
- 支持嵌套
- `+=` 运算符拼接数组

### 字典（Dict）

```vora
let d = {name: "Vora", version: 1}

// 下标读写
d["year"] = 2026
print(d["name"])            // → "Vora"

// Dict + Dict 合并（右侧覆盖同名键）
let a = {x: 1, y: 2}
let b = {y: 99, z: 3}
let c = a + b               // → {x:1, y:99, z:3}

// 嵌套字典
let nd = {outer: {inner: 42}}
print(nd["outer"]["inner"]) // → 42

// for-in 遍历键
let f = {a: 1, b: 2, c: 3}
let keys = []
for k in f { keys.add(k) }
```

- 字面量语法：`{key: value, ...}`，空字典用 `{}`
- 键类型：字符串（数字键自动转为字符串）
- 值类型：任意 Vora 值
- 下标 `[]` 读写（不存在的键 throw）
- 内建方法：`.keys()` → 键数组、`.values()` → 值数组、`.has(key)` → bool、`.remove(key)` → 删除并返回旧值
- `+` 运算符合并字典（右侧覆盖同名键）
- for-in 遍历字典键

### 对象系统

```vora
Obj Student(name, age) {
    this.name = name
    this.age = age

    func greet() {
        print("Hi, I'm ${this.name}, age ${this.age}")
    }
}

let s = Student("Alice", 20)
s.greet()                   // 输出: Hi, I'm Alice, age 20
print(s.name)               // 输出: Alice
s.age = 21                  // 属性赋值
```

- 构造函数（参数列表定义在类名后）
- 属性通过 `this.prop = value` 在构造函数中设置
- 方法定义在类体中
- 方法调用自动绑定 `this`
- 支持属性读写

#### 继承

```vora
Obj Animal(name) {
    this.name = name
    func speak() { print("...") }
}

Obj Dog : Animal (name, breed) {
    this.breed = breed
    func speak() { print("Woof! I'm " + this.name) }
}

Obj Puppy : Dog (name, breed, toy) {
    this.toy = toy
}

let d = Dog("Rex", "Husky")
d.speak()               // "Woof! I'm Rex" (子类覆盖)
print(d.name)           // "Rex" (父类构造函数设置的属性)
d.getName()             // 继承自 Animal 的方法
```

- 语法：`Obj Child : Parent (params) { ... }`
- 构造函数链：父类构造先执行（从根到子），参数按索引传递
- 方法查找沿继承链：子类 → 父类 → 父类的父类…
- 方法覆盖：子类同名方法覆盖父类方法
- 多层继承：`A → B → C` 支持任意深度

#### 多继承（C3 线性化）

```vora
Obj A() { func who() { return "A" } }
Obj B : A () { func who() { return "B" } }
Obj C : A () { func who() { return "C" } }

Obj D : B, C () {
    // MRO: D, B, C, A  (B 列前，冲突时 B 胜出)
}

let d = D()
d.who()          // → "B" (第一个父类方法优先)
d.fromA()        // → 从 A 继承 (祖父类方法仍可访问)
```

- 语法：`Obj Child : Parent1, Parent2 (params) { ... }`
- C3 线性化算法（Python 风格 MRO）：`L[C(B1,...,Bn)] = [C] + merge(L[B1],...,L[Bn], [B1,...,Bn])`
- 方法查找沿 MRO 链线性遍历（不重复搜索）
- 父类构造依次按 MRO 逆序执行（根最先）
- MRO 使用 `weak_ptr` 存储，防止继承链循环引用泄漏

#### super 关键字

```vora
Obj Dog : Animal(name, breed) {
    func speak() {
        return super.speak() + " woof!"   // 调用 Animal.speak()
    }
}
```

- 语法：`super.method()`，必须在方法体或构造函数内使用
- 在 MRO 链中从当前方法所属类之后开始查找
- 支持多层继承：`Puppy → Dog → Animal`，super 链式调用
- 与多继承结合：super 沿 C3 MRO 链查找

### 异常处理

#### try / catch / finally

```vora
try {
    let x = 1 / 0
} catch (e) {
    print("Error: ${e}")       // e 为错误消息字符串
} finally {
    print("Done")              // finally 始终执行
}
```

- `try/catch`、`try/finally`、`try/catch/finally` 三种组合
- `finally` 块始终执行（正常 / 异常 / return / break / continue）
- 捕获内建错误时，`e` 绑定为错误消息字符串

#### throw 与自定义错误

```vora
// 抛出自定义错误对象
Obj ValidationError(msg) {
    this.message = msg
}

throw ValidationError("Invalid input")

// 也可以直接抛出字符串
throw "something went wrong"
```

- `throw <expr>` 可抛出任意 Vora 值（字符串、对象、数字等）
- 自定义错误通过 `Obj` 定义，`throw` 抛出实例
- `catch (e)` 同时捕获用户 `throw` 的值（绑定原值）和内建 `RuntimeError`（绑定为字符串）

### 字节码 VM 模式

```bash
vora file.va
```

字节码编译器 + 栈式虚拟机是**默认**执行后端：

```bash
# 正常运行（VM 模式，默认）
vora file.va

# 字节码反汇编输出
vora file.va --tokens

# 切换到树遍历解释器
vora file.va --interpreter
```

VM 模式现已支持 Phase 1-4 全部功能，并包含多项性能优化：

**Phase 1（已完成）**：表达式、全局变量、if/else、while、三元运算符、数组、索引、字符串及内建函数调用（print、clock、assert、int、float、range、input、bin、oct、hex）。

**Phase 2（已完成）**：局部变量、块作用域、break/continue、for-in 循环（数组/字符串/range）、函数声明、基本 try/catch/throw 异常处理。

**Phase 3（已完成）**：对象（Obj）、属性读写、方法调用、继承（含多继承 C3 线性化）、for-in 对象属性迭代。

**Phase 4（已完成）**：VM 闭包（内层函数捕获外层变量的 upvalue 机制）、字符串插值（`${...}` 编译时解析为字节码拼接）、运行时错误 catch 完善（所有运行时错误优先走 catch 处理器）、Dict 字典类型。

**性能优化（v0.9）**：
- 全局变量整数驻留（parallel arrays 替代 `unordered_map<string, Value>`，O(1) 槽位访问，消除运行时字符串哈希）
- 快速数值操作码（`OP_SUB_NN`/`OP_MUL_NN` 等 8 条专用指令，跳过类型检查）
- 编译时常量折叠（字面量算术/比较/一元运算在编译期求值）
- 栈操作移动语义（`push`/`pop` 使用 `std::move`，`peek()` 返回引用）

**异常处理完善（v0.10）**：
- `finally` 块内 `break`/`continue`/`return` 正确路由：编译器捕获 try 体内的非局部跳转，字节码级重放到 finally 块后跳至原目标
- `OP_FINALLY_END` 异常重抛：通过 `exceptionInFlight` 标志位，finally 结束后自动将未捕获异常向外层 handler 传播
- `OP_CLEAR_EXCEPTION` 操作码：catch 块入口清除异常标志，防止 catch 后又重抛
- `throwException()` 不再弹出 handler，由目标处的 `OP_POP_CATCH` 统一弹出，避免嵌套 try 的 handler 错位
- `try` 内 `break`/`continue`/`return` 自动发出 `OP_POP_CATCH` 清理 catch handler 栈
- `for-in` 循环内 `break` 通过 `extraLocalsToPop` 清理 `_iter`/`_i`/`_len` 脱糖局部变量
- 常量池 `null` 去重优化

**多架构构建 + 应用图标 + 打包（v0.11）**：
- CMakePresets：20 个预设覆盖 Windows(x64/x86/arm64)、Linux(x64/x86/aarch64/armhf)、macOS(universal) × Debug/Release
- 交叉编译工具链：`cmake/toolchains/` 下 6 个文件（linux-i386、linux-aarch64、linux-armhf、windows-i386、windows-arm64、macos-universal）
- CPack 打包：`.msi` (WiX) / `.zip` (Windows)、`.deb` / `.rpm` / `.pkg.tar.xz` (Linux)、`.tar.gz` (macOS)
- 构建脚本：`build.ps1` (-Architecture/-Config/-Package)、`build.sh` (-a/-c/-p) 跨平台一键构建
- CI：GitHub Actions 多架构矩阵（Linux 4arch、Windows 3arch、macOS universal）× Debug/Release
- 应用图标：`vora.ico`（Windows）、`vora.png`/`vora_logo.svg`（Linux）、`vora.iconset/`（macOS）
- WiX 安装程序：64-bit .msi，用户可选安装目录，自动注册系统 PATH，控制面板卸载
- `vora.cmd`：小写启动器，终端直接输入 `vora` 运行

**运行时错误调用栈（v0.14）**：
- 异常时自动捕获完整调用栈：`captureStackTrace()` 遍历 CallFrame 链，解码 RLE 压缩行列信息
- 输出格式：`Stack trace (most recent call last):` → `[行:列] in <函数名>`
- 调用栈在异常抛出前快照（pre-unwind），避免栈帧弹出后信息丢失
- catch 捕获后自动清除调用栈，未捕获异常打印完整回溯

**C++ 单元测试系统（v0.13）**：
- 基于 doctest v2.4.11，6 个测试模块覆盖 value / chunk / lexer / parser / compiler / vm
- `vora_lib` OBJECT 库——Vora.exe 与 Vora_tests.exe 零重复编译
- CMakeLists.txt 中 6 个 CTest 注册，按模块前缀过滤（`unit-lexer`、`unit-compiler` 等）

**VM 栈动态扩容 + 常量池 16 位（v0.16）**：
- VM 栈：`std::vector` 按需增长，初始 reserve(1024)，突破旧 1024 硬限制，支持数千层递归
- 常量池：索引从 uint8 → uint16，新增 `OP_CONSTANT_LONG` 操作码，上限 256→65535

**错误信息增强（v0.17）**：
- 编译和运行时错误均显示出错的源码行 + 插入符 `^` 位置标记
- `printSourceLine()` 接收源文本和行列号，定位对应行并打印下划线标记
- 格式：`Error at [行:列]: 消息` + 源码行 + `     ^~~~~^`

### UTF-8 BOM 处理

自动去除源文件开头的 UTF-8 BOM（`EF BB BF`）。

## 当前限制

| 限制 | 说明 |
|------|------|
| 无模块系统 | 无法 `import` 其他文件 |
| 无标准库 | `std/` 目录为空 |
| 无不等于 null 的语法 | 需要 `x == null` |
| 无尾调用优化 (TCO) | 深度递归可能栈溢出（但有动态栈扩容，可承载数千层递归） |
| 常量池上限 65535 条目 | 16-bit 索引（OP_CONSTANT_LONG），超限报错退出 |
| 引用计数 GC | 使用 `shared_ptr`，`weak_ptr` 已切断类继承环，但用户创建的循环引用仍可能泄漏 |
| 无调试器 | 无可视化断点/单步执行 |

## 特性

| 特性 | 说明 |
|------|------|
| 类型标注不强制 | `:int`/`:float` 仅作文档，无编译时检查 |
| int64 / float64 双数值类型 | 整数字面量自动解析为 int64；自动提升规则 |
| Dict 字典类型 | `{key: val}` 字面量，下标 `[]` 读写，4 个内建方法 |
| `type()` / `len()` 内建函数 | 运行时类型查询（含 `"dict"`）+ 数组/字符串/字典长度 |
| 默认参数 | `func f(a, b = 1)`，默认值可引用前置形参 |
| C3 多继承 | `Obj D : B, C ()`，C3 线性化 MRO，构造链逆序执行 |
| `super` 关键字 | 方法体中调用父类方法（沿 MRO 查找） |
| for-in 全覆盖 | 数组、字符串、range、字典、对象遍历 |
| 运行时调用栈回溯 | 异常时打印 `Stack trace`，含文件/行号/列号/函数名 |
| 错误源码行 + 插入符 | 编译和运行时错误显示源码行 + `^` 位置标记 |
| VM 栈动态扩容 | `std::vector` 按需增长，支持数千层递归 |
| 常量池 16 位索引 | uint16 索引，上限 65535 条目 |
| 跨平台 | Windows / Linux / macOS，支持交叉编译 |
| 多架构 | x64, x86, arm64/aarch64, armhf |
| 原生打包 | .msi / .deb / .rpm / .pkg.tar.xz |
| C++ 单元测试 | doctest 6 模块，~100+ 用例 |
