## 语言特性总览

### 数据类型

| 类型 | Vora 字面量 | 内部表示 | 示例 |
|------|------------|----------|------|
| Null | `null` | `std::nullptr_t` | `let x = null` |
| Number | 数字 | `double` | `42`, `3.14` |
| Boolean | `TRUE` / `FALSE` | `bool` | `let flag = TRUE` |
| String | 双引号字符串 | `std::string` | `"hello"` |
| Array | `[...]` | `shared_ptr<Array>` | `[1, 2, 3]` |
| Function | `func` 声明 | `shared_ptr<VoraFunction>` | `func f() {}` |
| Object | `Obj` 声明 | `shared_ptr<ObjectInstance>` | `Obj Foo() {}` |

### 变量系统

```vora
let x = 10              // 声明并初始化
x = 20                  // 重新赋值
```

- 使用 `let` 关键字声明
- 支持块级作用域
- 变量沿作用域链向上查找

### 运算符

| 分类 | 运算符 | 说明 |
|------|--------|------|
| 算术 | `+`, `-`, `*`, `/`, `%` | 加、减、乘、除、取模 |
| 幂运算 | `**` | 右结合 |
| 比较 | `<`, `<=`, `>`, `>=` | 关系比较 |
| 相等 | `==`, `!=` | 相等/不等 |
| 逻辑 | `&&`, `\|\|`, `!` | 与、或、非 |
| 自增 | `++`, `--` | Token 已定义，语义未实现 |
| 赋值 | `=` | 变量赋值和属性赋值 |

### 字符串插值

```vora
let name = "World"
print("Hello ${name}!")          // 输出: Hello World!

Obj Person(n) { this.name = n }
let p = Person("Alice")
print("Hi ${p.name}")            // 输出: Hi Alice
```

- 语法：`${variable}` 或 `${obj.property}`
- 支持多级属性访问
- 自动类型转换（数字、布尔、null → 字符串）
- 变量未找到时保留原始 `${...}` 文本

### 控制流

#### if / else

```vora
if (x > 0) {
    print("positive")
} else if (x < 0) {
    print("negative")
} else {
    print("zero")
}
```

#### while 循环

```vora
let i = 0
while (i < 10) {
    print(i)
    i = i + 1
}
```

#### for-in 循环

```vora
for item in [1, 2, 3, 4, 5] {
    print(item)
}
```

- 仅支持数组遍历
- 每次迭代创建新作用域

### 函数

```vora
func add(a, b) {
    return a + b
}

let result = add(1, 2)    // result = 3
```

- 一等公民（可作为值传递）
- 闭包支持（捕获定义时的环境）
- 参数个数校验
- 通过 `return` 返回值（无 return 则返回 null）

### 数组（未实现）

```vora
let arr = [1, "hello", TRUE, null]
print(arr[0])              // 输出: 1
print(arr[1])              // 输出: hello
```

- 动态类型元素
- 整数索引（从 0 开始）
- 越界检查
- 支持嵌套

### 对象系统

```vora
Obj Student(name, age) {
    this.name = name
    this.age = age

    func greet() {
        print("Hi, I'm ${this.name}, age ${this.age}")
    }
}

let s = Student("Alice", 20)
s.greet()                   // 输出: Hi, I'm Alice, age 20
print(s.name)               // 输出: Alice
s.age = 21                  // 属性赋值
```

- 构造函数（参数列表定义在类名后）
- 属性通过 `this.prop = value` 在构造函数中设置
- 方法定义在类体中
- 方法调用自动绑定 `this`
- 支持属性读写

### 内建函数

| 函数 | 签名 | 说明 |
|------|------|------|
| `print` | `print(...)` | 变参输出，空格分隔，末尾换行 |
| `clock` | `clock()` | 返回当前时间戳（秒，毫秒精度） |
| `input` | `input()` | 从标准输入读取一行，返回字符串 |

### 注释

```vora
// 这是行注释

/* 这是
   块注释 */
```

- 不支持嵌套块注释

### 语句分隔符

- 分号 `;` 可选——不强制要求
- 块内语句通过换行自然分隔

## 开发者工具

### REPL 模式

```bash
vora --repl
```

- 交互式读取-求值-打印循环
- 表达式自动打印结果
- 持久化环境（变量跨行保持）
- 错误不中断会话

### AST 打印

```bash
vora file.va --ast-printer
```

输出 S-expression 格式的 AST：
```
(program (let x (+ 1 2)) (print x))
```

### Token 流打印

```bash
vora file.va --tokens
```

输出 Token 序列：
```
LET(let) IDENTIFIER(x) EQUAL(=) NUMBER(10) SEMICOLON(;) EOF()
```

### UTF-8 BOM 处理

自动去除源文件开头的 UTF-8 BOM（`EF BB BF`）。

## 当前限制

| 限制 | 说明 |
|------|------|
| 无 `break` / `continue` | 循环中无法提前跳出或跳过 |
| 无异常处理 | 没有 `try/catch/finally` |
| 无模块系统 | 无法 `import` 其他文件 |
| 无标准库 | `std/` 目录为空 |
| 无测试 | `tests/` 目录为空 |
| 无数值运算重载 | `+` 不能用于字符串拼接 |
| 无数组方法 | 无 `push`、`pop`、`length` 等 |
| 无字符串方法 | 无 `length`、`substring` 等 |
| for 仅支持数组 | 无法遍历字符串、范围等 |
| 无整数类型 | 所有数字统一为 double |
| 无类继承 | `Obj` 无继承机制 |
| 无不等于 null 的语法 | 需要 `x == null` |
| 无错误恢复 | 解析错误立即终止 |
