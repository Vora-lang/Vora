## 语言特性总览

### 数据类型

| 类型 | Vora 字面量 | 内部表示 | 示例 |
|------|------------|----------|------|
| Null | `null` | `std::nullptr_t` | `let x = null` |
| Number | 数字 | `double` | `42`, `3.14`, `0xFF`, `0o77`, `0b101` |
| Boolean | `TRUE` / `FALSE` | `bool` | `let flag = TRUE` |
| String | 双引号字符串 | `std::string` | `"hello"` |
| Array | `[...]` | `shared_ptr<Array>` | `[1, 2, 3]` |
| Function | `func` 声明 | `shared_ptr<VoraFunction>` | `func f() {}` |
| Object | `Obj` 声明 | `shared_ptr<ObjectInstance>` | `Obj Foo() {}` |

### 变量系统

```vora
let x = 10              // 声明并初始化
x = 20                  // 重新赋值
```

- 使用 `let` 关键字声明
- 支持块级作用域
- 变量沿作用域链向上查找

### 运算符

| 分类 | 运算符 | 说明 |
|------|--------|------|
| 算术 | `+`, `-`, `*`, `/`, `%` | 加、减、乘、除、取模 |
| 幂运算 | `**` | 右结合 |
| 比较 | `<`, `<=`, `>`, `>=` | 关系比较 |
| 相等 | `==`, `!=` | 相等/不等 |
| 逻辑 | `&&`, `||`, `!` | 与、或、非 |
| 自增/自减 | `++`, `--` | 前缀/后缀均支持 |
| 复合赋值 | `+=`, `-=`, `*=`, `/=`, `%=` | 加法/减法/乘法/除法/取模赋值 |
| 三元 | `? :` | 条件三元运算符，短路求值，右结合 |
| 赋值 | `=` | 变量赋值和属性赋值 |

### 字符串插值

```vora
let name = "World"
print("Hello ${name}!")          // 输出: Hello World!

Obj Person(n) { this.name = n }
let p = Person("Alice")
print("Hi ${p.name}")            // 输出: Hi Alice
```

- 语法：`${variable}` 或 `${obj.property}`
- 支持多级属性访问
- 自动类型转换（数字、布尔、null → 字符串）
- 变量未找到时保留原始 `${...}` 文本

### 字符串拼接与索引

```vora
let s = "Hello" + " " + "World"  // 字符串拼接
print(s[0])                       // 索引访问 → "H"
print(s[3])                       // → "l"
```

- `+` 运算符支持 `string + string` 拼接
- `[]` 索引访问单个字符（返回字符串）
- 字符串可用单引号或双引号

### 类型标注

```vora
let x:int = 42                   // 标注期望 int 类型
let y:float = 3.14               // 标注期望 float 类型
let z = int(input("> "))         // 类型转换 + 输入
```

- `:int` / `:float` 标注（当前仅作文档用途，不强制校验）
- 配合 `int()` / `float()` 进行显式类型转换

### 控制流

#### if / else

```vora
if (x > 0) {
    print("positive")
} else if (x < 0) {
    print("negative")
} else {
    print("zero")
}
```

#### 三元运算符 `? :`

```vora
let result = x > 0 ? "positive" : "negative"

// 右结合：a ? b : c ? d : e  →  a ? b : (c ? d : e)
let grade = score > 90 ? "A" : score > 80 ? "B" : "C"

// 短路求值：只求值被选中的分支
let val = condition ? expensiveOp1() : expensiveOp2()
```

#### while 循环

```vora
let i = 0
while (i < 10) {
    print(i)
    i += 1                       // 复合赋值
}
```

- 支持 `break` / `continue`

#### for-in 循环

```vora
// 数组
for item in [1, 2, 3, 4, 5] {
    print(item)
}

// 字符串
for ch in "Vora" {
    print(ch)
}

// range
for i in range(5) {
    print(i)
}
```

- 支持数组、字符串、`range()` 结果
- 每次迭代创建新作用域
- 支持 `break` / `continue`

### 函数

```vora
func add(a, b) {
    return a + b
}

let result = add(1, 2)    // result = 3
```

- 一等公民（可作为值传递）
- 闭包支持（捕获定义时的环境）
- 参数个数校验
- 通过 `return` 返回值（无 return 则返回 null）

### 数组

```vora
let arr = [1, "hello", TRUE, null]
print(arr[0])              // 输出: 1
print(arr[1])              // 输出: hello
arr += [3, 4]              // 数组拼接

for item in arr {          // 遍历
    print(item)
}
```

- 动态类型元素
- 整数索引（从 0 开始）
- 越界检查
- 支持嵌套
- `+=` 运算符拼接数组

### 对象系统

```vora
Obj Student(name, age) {
    this.name = name
    this.age = age

    func greet() {
        print("Hi, I'm ${this.name}, age ${this.age}")
    }
}

let s = Student("Alice", 20)
s.greet()                   // 输出: Hi, I'm Alice, age 20
print(s.name)               // 输出: Alice
s.age = 21                  // 属性赋值
```

- 构造函数（参数列表定义在类名后）
- 属性通过 `this.prop = value` 在构造函数中设置
- 方法定义在类体中
- 方法调用自动绑定 `this`
- 支持属性读写

#### 继承

```vora
Obj Animal(name) {
    this.name = name
    func speak() { print("...") }
}

Obj Dog : Animal (name, breed) {
    this.breed = breed
    func speak() { print("Woof! I'm " + this.name) }
}

Obj Puppy : Dog (name, breed, toy) {
    this.toy = toy
}

let d = Dog("Rex", "Husky")
d.speak()               // "Woof! I'm Rex" (子类覆盖)
print(d.name)           // "Rex" (父类构造函数设置的属性)
d.getName()             // 继承自 Animal 的方法
```

- 语法：`Obj Child : Parent (params) { ... }`
- 构造函数链：父类构造先执行（从根到子），参数按索引传递
- 方法查找沿继承链：子类 → 父类 → 父类的父类…
- 方法覆盖：子类同名方法覆盖父类方法
- 多层继承：`A → B → C` 支持任意深度

### 异常处理

#### try / catch / finally

```vora
try {
    let x = 1 / 0
} catch (e) {
    print("Error: ${e}")       // e 为错误消息字符串
} finally {
    print("Done")              // finally 始终执行
}
```

- `try/catch`、`try/finally`、`try/catch/finally` 三种组合
- `finally` 块始终执行（正常 / 异常 / return / break / continue）
- 捕获内建错误时，`e` 绑定为错误消息字符串

#### throw 与自定义错误

```vora
// 抛出自定义错误对象
Obj ValidationError(msg) {
    this.message = msg
}

throw ValidationError("Invalid input")

// 也可以直接抛出字符串
throw "something went wrong"
```

- `throw <expr>` 可抛出任意 Vora 值（字符串、对象、数字等）
- 自定义错误通过 `Obj` 定义，`throw` 抛出实例
- `catch (e)` 同时捕获用户 `throw` 的值（绑定原值）和内建 `RuntimeError`（绑定为字符串）

### 内建函数

| 函数 | 签名 | 说明 |
|------|------|------|
| `print` | `print(...)` | 变参输出，空格分隔，末尾换行 |
| `clock` | `clock()` | 返回当前时间戳（秒，毫秒精度） |
| `input` | `input(prompt?)` | 读取一行输入，可选 prompt 参数 |
| `int` | `int(value)` | 转换为整数（截断小数）；接受 number/string/bool |
| `float` | `float(value)` | 转换为浮点数；接受 number/string/bool |
| `range` | `range(stop)` / `range(start, stop)` / `range(start, stop, step)` | 生成数字数组 |
| `assert` | `assert(condition, message?)` | 断言条件为真，否则抛出错误 |
| `bin` | `bin(num)` | 数字 → 二进制字符串 `"0b..."` |
| `oct` | `oct(num)` | 数字 → 八进制字符串 `"0o..."` |
| `hex` | `hex(num)` | 数字 → 十六进制字符串 `"0x..."` |

### 注释

```vora
// 这是行注释

/* 这是
   块注释 */

/* 外层
   /* 内层 */
   仍在注释中 */
```

- 支持嵌套块注释

### 语句分隔符

- 分号 `;` 可选——不强制要求
- 块内语句通过换行自然分隔

## 开发者工具

### REPL 模式

```bash
vora --repl
```

- 交互式读取-求值-打印循环
- 表达式自动打印结果
- 持久化环境（变量跨行保持）
- 错误不中断会话

### AST 打印

```bash
vora file.va --ast-printer
```

输出 S-expression 格式的 AST：
```
(program (let x (+ 1 2)) (print x))
```

### Token 流打印

```bash
vora file.va --tokens
```

输出 Token 序列：
```
LET(let) IDENTIFIER(x) EQUAL(=) NUMBER(10) SEMICOLON(;) EOF()
```

### 字节码 VM 模式

```bash
vora file.va
```

字节码编译器 + 栈式虚拟机是**默认**执行后端：

```bash
# 正常运行（VM 模式，默认）
vora file.va

# 字节码反汇编输出
vora file.va --tokens

# 切换到树遍历解释器
vora file.va --interpreter
```

VM 模式现已支持 Phase 1-4 全部功能：

**Phase 1（已完成）**：表达式、全局变量、if/else、while、三元运算符、数组、索引、字符串及内建函数调用（print、clock、assert、int、float、range、input、bin、oct、hex）。

**Phase 2（已完成）**：局部变量、块作用域、break/continue、for-in 循环（数组/字符串/range）、函数声明、基本 try/catch/throw 异常处理。

**Phase 3（已完成）**：对象（Obj）、属性读写、方法调用、继承（含多层继承）。

**Phase 4（已完成）**：VM 闭包（内层函数捕获外层变量的 upvalue 机制）、字符串插值（`${...}` 编译时解析为字节码拼接）、运行时错误 catch 完善（所有运行时错误优先走 catch 处理器）。

### UTF-8 BOM 处理

自动去除源文件开头的 UTF-8 BOM（`EF BB BF`）。

## 当前限制

| 限制 | 说明 |
|------|------|
| 无模块系统 | 无法 `import` 其他文件 |
| 无标准库 | `std/` 目录为空 |
| 无数组方法 | 无 `push`、`pop`、`length` 等 |
| 无字符串方法 | 无 `length`、`substring` 等 |
| 无不等于 null 的语法 | 需要 `x == null` |
| 解析器不支持 `arr[0] = 10` | 索引赋值语法暂不支持（非 VM 问题） |
| VM `++/--` 后缀行为 | 后缀 `x++` 在 VM 中等效于前缀 `++x` |

## 特性

| 特性 | 说明 |
|------|------|
| 类型标注不强制 | `:int`/`:float` 仅作文档，无编译时检查 |
| 无整数类型 | 所有数字统一为 double |
