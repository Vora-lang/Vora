## 各文件详细说明

### 入口文件

#### `src/main.cpp`（~420 行）

程序入口点，负责：

- **命令行参数解析**：支持 `--repl`、`--ast-printer`、`--tokens`、`--interpreter`、文件路径
- **文件读取**：`readFile()` 读取 `.va` 源文件，并去除 UTF-8 BOM 头
- **脚本执行**：`runScript()` 串联 Lexer → Parser → Compiler → VM（默认）或 Interpreter（`--interpreter`）
- **REPL 模式**：`runREPL()` 实现交互式读取-求值-打印循环（仅解释器模式）
- **全局初始化**：VM 模式下通过 `vm.initGlobals()` 预分配编译器驻留的全局变量槽位，并注册 14 个内建函数（含 `len()`/`type()`）
- **错误处理**：捕获 `RuntimeError` 异常并输出行号和错误信息

### 语法分析器（Parser）

#### `src/parser/parser.h`（71 行）

**类 `Parser`**：Pratt 解析器（优先级爬升法）。

| 成员 | 类型 | 说明 |
|------|------|------|
| `tokens` | `vector<Token>` | Token 流 |
| `current` | `size_t` | 当前 Token 索引 |

**公开方法**：
- `Parser(vector<Token>)`：构造函数
- `parse()` → `unique_ptr<Program>`：解析入口，返回完整 AST

**私有方法 — 表达式解析**：
- `expression()` → `Expr`：表达式入口（调用 `parsePrecedence(0)`）
- `parsePrecedence(int)` → `Expr`：Pratt 核心 —— 按优先级解析
- `primary()` → `Expr`：解析基础表达式（字面量、变量、数组、分组、一元运算）
- `call()` → `Expr`：解析调用链（函数调用、索引访问、属性访问）
- `finishCall(Expr callee)` → `Expr`：完成函数调用解析

**私有方法 — 语句解析**：
- `statement()` → `Stmt`：语句分发入口
- `letStatement()` → `Stmt`：`let` 声明
- `blockStatement()` → `Stmt`：`{ }` 代码块
- `returnStatement()` → `Stmt`：`return` 语句
- `ifStatement()` → `Stmt`：`if/else` 分支
- `whileStatement()` → `Stmt`：`while` 循环
- `forStatement()` → `Stmt`：`for-in` 循环
- `funcStatement()` → `Stmt`：`func` 函数声明
- `objStatement()` → `Stmt`：`Obj` 对象声明

**私有方法 — 工具**：
- `isAtEnd()` / `advance()` / `peek()` / `previous()`：Token 流导航
- `match(TokenType)` / `check(TokenType)`：Token 匹配
- `getPrecedence(TokenType)` → `int`：查询运算符优先级

**优先级表**：

| 优先级 | 运算符 | 结合性 |
|--------|--------|--------|
| 1 | `=`, `||` | 右结合（`=`） |
| 2 | `&&` | 左结合 |
| 3 | `==`, `!=` | 左结合 |
| 4 | `<`, `<=`, `>`, `>=` | 左结合 |
| 5 | `+`, `-` | 左结合 |
| 6 | `*`, `/`, `%` | 左结合 |
| 7 | `**` | 右结合 |

#### `src/parser/parser.cpp`（1114 行）

Parser 的完整实现。表达式解析使用 Pratt 算法，语句解析使用递归下降。赋值（`=`）在 Pratt 中统一处理，通过 `dynamic_cast` 判断左侧是 `VariableExpr` 还是 `PropertyExpr`，分别生成 `AssignmentExpr` 或 `PropertyAssignmentExpr`。支持 `break`/`continue`/`throw`/`try`、三元运算符 `? :`、自增/自减、字符串插值等。
